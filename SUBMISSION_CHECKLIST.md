## 🚀 How to Run the Project

### 1. Start Backend
```bash
cd ml-backend
pip install -r requirements.txt
uvicorn main:app --reload --port 8000
```

### 2. Start Frontend
```bash
npm install
npm run dev
```

### 3. Access Application
- Frontend: http://localhost:8080
- Backend API: http://localhost:8000
